## 编程
### 学习如何创建窗口

|语句|含义|参数|返回值|样例|
|:---:|:----:|:---:|:-----:|:--:|
|cv2.namedWindow()|创建一个窗口|第一个参数为窗口的名字，第二个参数为窗口的类型|void|cv2.namedWindow('new window',cv2.WINDOW_NORMAL)|
|cv2.resizeWindow(,,)|重新设置窗口的大小|窗口名字，宽，高|void|cv2.resizeWindow('new window',640,480)|
|cv2.imshow(,)|将图像展示到特定的窗口|窗口名字，图像数据|void|cv2.imshow('new window',0)|
|cv2.waitKey()|等待按键事件发生或者特定的时间|0就是永久，正数就是时间(ms)|-1(规定时间到了)或者按下的按键|key=cv2.waitKey(0)（将按下的按键赋值给key）|
|cv2.destroyAllWindows()|关闭所有窗口，释放内存|Null|void|cv2.destroyAllWindows()|
|cv2.destroyWindow()|关闭这个窗口，释放内存|窗口名字|void|cv2.destroyWindow()|

#### 代码

```python
import cv2
cv2.namedWindow('new window',cv2.WINDOW_NORMAL)
cv2.resizeWindow('new window',640,480)
cv2.imshow('new window',0)
while True:
    key=cv2.waitKey(0)
    if (key==ord('q')):
        print("q pressed")
        break
print(key)
cv2.destroyAllWindows()
```

### 学会如何显示图片

|语句|含义|参数|返回值|样例|
|:---:|:----:|:---:|:-----:|:--:|
|cv2.imread()|读取图片|图片的地址，图片的形式|mat|img=cv2.imread('/home/song/图片/壁纸2.jpeg',cv2.IMREAD_COLOR)|

#### 代码

```python
import cv2
cv2.namedWindow('WindowsForImages',cv2.WINDOW_NORMAL)
img=cv2.imread('/home/song/图片/壁纸2.jpeg',cv2.IMREAD_COLOR)
cv2.imshow('WindowsForImages',img)
while True:
    key=cv2.waitKey(0)
    if key==ord("q"):
        print("P Pressed!")
        break
cv2.destroyWindow('WindowsForImages')
```

### 学会使用grep查找
   >grep -rin --include"*.XXX" "XXXX" ___.___
   - -r：递归搜索
   - -i：忽略大小写
   - -n:显示行号
   - --include""是所查找文件名要包含的内容
   - *.XXX是扩展名是XXX
   - "XXXX"是要搜索的内容
   - ***.*** 意味着当前目录
  ```bash
  grep -rin --include="*.hpp" "imread(" .
  ```
