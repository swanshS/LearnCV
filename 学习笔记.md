#### 学习如何创建窗口

|语句|含义|参数|返回值|样例|
|:---:|:----:|:---:|:-----:|:--:|
|cv2.namedWindow()|创建一个窗口|第一个参数为窗口的名字，第二个参数为窗口的类型|void|cv2.namedWindow('new window',cv2.WINDOW_NORMAL)|
|cv2.resizeWindow(,,)|重新设置窗口的大小|窗口名字，宽，高|void|cv2.resizeWindow('new window',640,480)|
|cv2.imshow(,)|将图像展示到特定的窗口|窗口名字，图像数据|void|cv2.imshow('new window',0)|
|cv2.waitKey()|等待按键事件发生或者特定的时间|0就是永久，正数就是时间(ms)|-1(规定时间到了)或者按下的按键|key=cv2.waitKey(0)（将按下的按键赋值给key）|
|cv2.destroyAllWindows()|关闭所有窗口，释放内存|Null|void|cv2.destroyAllWindows()|
|cv2.destroyWindow()|关闭这个窗口，释放内存|窗口名字|void|cv2.destroyWindow()|

##### 代码

```python
import cv2
cv2.namedWindow('new window',cv2.WINDOW_NORMAL)
cv2.resizeWindow('new window',640,480)
cv2.imshow('new window',0)
while True:
    key=cv2.waitKey(0)
    if (key==ord('q')):
        print("q pressed")
        break
print(key)
cv2.destroyAllWindows()
```

#### 从摄像头设备或者本地视频文件中捉取视频帧

|语句|含义|参数|返回值|样例|
|:---:|:----:|:---:|:-----:|:--:|
|cv2.VideoCapture()|打开摄像头获得资源|设备号，摄像头资源的底层接口|返回一个类，这个语句一般赋值给一个变量，之后通过变量来进行操作|capture=cv2.VideoCapture(0,cv2.CAP_ANY)|
|cv2.VideoCapture()|打开摄像头获得资源|本地视频的地址，摄像头资源的底层接口|返回一个类，这个语句一般赋值给一个变量，之后通过变量来进行操作|capture=cv2.VideoCapture('/run/media/song/Data/UESTC/课程/2025.1eagp/Academic English Materials PHASE 1-20250917/CS3/CS3 Talking with your Tutor - Part 1.mp4',cv2.CAP_ANY)|

#### 根据视频帧制作视频文件

|语句|含义|参数|返回值|样例|
|:---:|:----:|:---:|:-----:|:--:|
|variable=cv2.VideoWriter.fourcc()|规定视频编码格式|字符串（编码格式）|类|fourcc=cv2.VideoWriter.fourcc('M','J','P','G')|
|variable=cv2.VideoWriter()|将视频文件的信息给类|地址，编码格式，帧数，分辨率，是否是彩色|类|vw=cv2.VideoWriter('Pic/MakeVideo.mp4',fourcc,25,(640,480),isColor=True)|
|variable.write(mat)|将视频帧写入|mat|void|vw.write(frame)|
|variable.release()|释放资源|null|void|vw.release()|

##### 代码
```python
import cv2

#创建窗口
cv2.namedWindow('video',cv2.WINDOW_NORMAL)

#写入视频编码格式
fourcc=cv2.VideoWriter.fourcc('M','J','P','G')

#获得为了视频打开摄像头，0是设备（摄像头）编号，CAP_ANY是为系统自动找到一个合适的API接口（哪一个底层的库来访问摄像头资源）
capture=cv2.VideoCapture(0,cv2.CAP_ANY)

#
vw=cv2.VideoWriter('Pic/MakeVideo.mp4',fourcc,25,(640,480),isColor=True)
#变量
a=0#保存图片数目
Camera_First=1#是否第一次获得摄像头资源
while capture.isOpened():#判断摄像头是否被打开
    #capture.read()两个返回值：第一个是是否成功读取帧，第二个是图片（mat格式u）
    ret,frame=capture.read()
    if ret==False:
        if Camera_First==1:
            print("Camera False!")
            Camera_First=0
    elif ret==True:
        if Camera_First==1:
            print("Camera OK!")
            Camera_First=0
            #显示图片
        cv2.imshow('video',frame)
        #保存到特定地址
        vw.write(frame)
        #等待key
        key=cv2.waitKey(1)
        #退出
        if key&0xFF==ord('q'):
            print("Quit the window!")
            break
        #保存
        elif key&0xFF==ord('s'):
            cv2.imwrite('Pic/Camera'+str(a)+'.png',frame)
            print("Picture Saved!")
            a=a+1

#释放资源
capture.release()
vw.release()
cv2.destroyAllWindows()
```

#### trackbar
|语句|含义|参数|返回值|样例|

import cv2
import numpy as np
def cb(x):
    pass

cv2.namedWindow('window',cv2.WINDOW_NORMAL)
cv2.resizeWindow('window',640,480)

#创建trackbar
cv2.createTrackbar('R','window',0,255,cb)

    g=cv2.getTrackbarPos('G','window')

    #赋值
    img[:]=[b,g,r]
    cv2.imshow('window',img)
    key=cv2.waitKey(1)
    if key==ord('q'):
        break

#### Numpy与图像
将一个 ***numpy*** 数组赋值给 ***img***，后续操作就 **跟普通的一样**
##### 代码
>这是一个通过鼠标左右建控制屏幕亮度（颜色）的代码
```python
import numpy as np
import cv2
color = 255
img = np.full((480,640),color,np.uint8)
cv2.namedWindow('NumpyWindow',cv2.WINDOW_AUTOSIZE)

def OnMouse(event,x,y,flag,para):
    global color
    #左键：亮度升高
    if event==cv2.EVENT_LBUTTONDOWN:
        color=(color+50+255)%255
    #右键：亮度降低
    elif event==cv2.EVENT_RBUTTONDOWN:
        color=(color+255-50)%255
    img = np.full((480,640),color,np.uint8)
    cv2.imshow('NumpyWindow',img)
cv2.setMouseCallback('NumpyWindow',OnMouse,param=None)
while True:
    cv2.imshow("NumpyWindow",img)
    key =cv2.waitKey(0)
    img = np.full((480,640),color,np.uint8)
    if key == ord('q'):
        break
cv2.destroyAllWindows()

```

#### 颜色通道

|语句|含义|参数|返回值|样例|
|:---:|:----:|:---:|:-----:|:--:|
|cv2.split()|将现有图像分割成三个通道|img|对应的通道的值|b,g,r=cv2.split(img)|
|cv2.merge((g,r,b))|将不同通道的值和在一起形成一个三通道的数组|（多个通道）|多通道数组|cv2.merge((g,r,b))|
##### 代码
```python
import cv2
import numpy

img=cv2.imread('./Pic/Pic.png')
cv2.namedWindow('Pic',cv2.WINDOW_NORMAL)
cv2.resizeWindow('Pic',480,640)
#分割通道
b,g,r=cv2.split(img)
while True:
    #合并通道
    cv2.imshow('Pic',cv2.merge((g,r,b)))
    key=cv2.waitKey(0)
    if key==ord('q'):
        break
cv2.destroyAllWindows()
```
